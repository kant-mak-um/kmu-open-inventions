# kmu-open-inventions

## Purpose
This repository contains **open reference designs and documentation** originating from **Kant‑Mak‑Um Design Academy**.

It is the public, version-controlled backbone for:
- Open inventions and reference artifacts
- Systems engineering documentation and configuration management
- Community-safe materials (no trade secrets)

## Authority & Governance
- **Design Origin:** Kant‑Mak‑Um Design Academy
- **Repository Role:** Open distribution of reference designs + SE documentation
- **Standards:** May be referenced by standards bodies (e.g., Modular Boat Alliance) *by tagged releases only*
- **Manufacturing:** Not included
- **Commercial Products:** Not included

## Scope (What’s Included)
This repository may include:
- ConOps (system narrative walkthroughs)
- System Requirements (top-level “shall” statements with acceptance criteria)
- Interface Control Documents (ICDs)
- Risk Register (top risks + mitigations)
- Configuration Management notes (design decisions, baselines, milestone tags)
- Reference geometry and interface definitions intended for public use

## Exclusions (What’s Never Included)
- Proprietary manufacturing parameters or process recipes
- Tooling designs that reveal manufacturing trade secrets
- Resin formulations, optimization data, or confidential test results
- Any advanced/licensable designs intended for private licensing

## Versioning
Semantic versioning is used for referenced releases:
MAJOR.MINOR.PATCH

Standards bodies and downstream users should reference **tagged releases only**.
